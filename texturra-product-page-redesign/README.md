# TEXURRA — Product Detail Redesign

Prototip HTML standalone pentru pagina de produs „Draperie Velvet 308 – Albastru-Verzui Pastel Blackout din Catifea de Lux”.

## Conținut

- `index.html` — prototip desktop + responsive, cu interacțiuni UI de concept.
- `DESIGN-SPEC-RO.md` — rationale de design și structură pe secțiuni.
- `AI-IMPLEMENTATION-PROMPT-RO.md` — prompt complet pentru implementare în proiectul real / Codex.
- `assets/` — imagini de concept folosite local în prototip.

## Cum îl deschizi

Deschide `index.html` într-un browser. Nu necesită build sau server.

## Elemente care trebuie conectate la aplicația reală

- date produs, cod produs, material, culori și prețuri;
- reguli de calcul pentru lățime, înălțime, confecționare și număr bucăți;
- stoc și disponibilitate;
- favorite, coș, cont, search;
- imaginile și logo-ul oficial Texturra;
- rutele reale și conținutul CMS.

## Notă

Este un concept UI/UX. Conținutul descriptiv, etichetele și prețurile sunt demo și trebuie mapate pe datele reale.
